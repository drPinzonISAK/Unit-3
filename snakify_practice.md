# Unit3 Snakify practices
## 1. Input, print and numbers ##
### Sum of three numbers ###
```
# This program reads three numbers and prints their sum:
a = int(input())
b = int(input())
c = int(input())
print(a + b + c)
```
### Square ###
```
a = int(input())
print(a ** 2)
```
### Apple Sharing ###
```
N = int(input())
K = int(input())
print(K // N)
print(K % N)
```
